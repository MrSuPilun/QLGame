```
PROJECT QLGAME

+---includes                 -> Thư mục chứa header và footer  
+---css                      -> Thư mục chứa file css  
+---js                       -> Thư mục chứa file js  
|       index.js

+---setup                    -> Thư mục chứa các file khởi tạo hệ thống
```
